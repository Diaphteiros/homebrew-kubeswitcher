# Kubeswitcher Homebrew Tap

## Installation

`brew install diaphteiros/kubeswitcher/<formula>`

Or `brew tap diaphteiros/kubeswitcher` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "diaphteiros/kubeswitcher"
brew "<formula>"
```
